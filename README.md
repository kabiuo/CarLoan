# About this CarLoan

### 这是仅使用了servlet的一个原生项目，也是在我学习Java web，比较熟悉的一种实现方式。

### 可以看到，在项目中采用了分层的形式：

> com.woniu.carloan
>> bean
>>> 这里放的都是对应数据库的表的实体类
>> dao
>>> dao接口类
>>> Impl
>>>> dao接口的实现类
>> servlet
>>> View与后台服务的服务连接器
>> service
>>> service服务接口类
>>> Impl
>>>> service服务实现类
>> util
>>> 与数据库连接的一些类
